# Advent of Rust 2018

> Another year and another advent of code

My implementation of the [Advent of Code](https://adventofcode.com/) puzzles for the 2019 year edition of course in [Rust](https://rust-lang.org/).

## Instructions

### Running all tests

```bash
cargo test
```

### Runnin a single day

```bash
cargo test day01
```
